    🎯 Introducción

    El módulo de Configuración permite establecer los parámetros fundamentales del sistema de gestión de recaudos, incluyendo información del conjunto, Líneas, módulos de cobro y configuraciones contables.

    💡 Objetivo: Configurar correctamente todos los parámetros del sistema para que funcione de acuerdo a las necesidades específicas de su Cartera de Crédito.

    Áreas de Configuración

    • 🏢 Información General: Datos básicos del conjunto

    • 💰 Configuración Contable: Cuentas y parámetros fiscales

    🚪 Cómo Acceder

    2. Navegue al módulo "Gestión de Recaudos"
    3. Haga clic en "Configuración PH" o "Configuración"

    • 🏪 Comercial: Centros comerciales, oficinas

    • 🌟 Genérico: Gestión general de recaudos

    Políticas de Recaudo (Mora y Descuento)

    El sistema permite gestionar de forma automatizada tanto los castigos por mora como los incentivos por pronto pago:

    • 🔥 Intereses de Mora: Se puede habilitar un interruptor global para cobrar intereses automáticos a las Préstamos en mora. Se parametriza el porcentaje mensual y el día límite de pago (después de este día se genera mora).
    • 💎 Descuento por Pronto Pago: Incentivo que reduce el valor a pagar si el Deudor cancela antes de cierta fecha.
      • Interruptor Global: Permite encender o apagar el descuento para toda la copropiedad con un solo clic.
      • Día Límite: El Deudor debe pagar la totalidad antes de este día para ganar el descuento.
      • Cuenta Contable: El descuento se registra en tiempo real como un Menor Valor del Ingreso (Ej: cuenta 4175) sin afectar el saldo histórico de la cartera.

    📋 Configuración de Módulos

    ¿Qué son los Módulos?

    Los módulos definen los diferentes tipos de servicios o características que pueden tener las Préstamos, determinando qué conceptos de cobro se les aplicarán.

    Módulos Típicos

    Módulo Descripción Conceptos Típicos
    🏠 Residencial Apartamentos y casas Cuota, Aseo, Vigilancia
    🏪 Comercial Locales y oficinas Cuota comercial, Publicidad
    🚗 Parqueadero Espacios de estacionamiento Cuota parqueadero, Mantenimiento
    📦 Depósito Bodegas y cuartos útiles Cuota depósito
    🌟 Especial Servicios adicionales Servicios premium, Amenidades

    Crear Nuevo Módulo

    1. Acceda a la sección "Módulos"
    2. Haga clic en "Nuevo Módulo"
    3. Defina nombre y descripción
    4. Configure los conceptos aplicables
    5. Guarde la configuración

    Asignación de Módulos

    Los módulos se asignan a las Préstamos para determinar qué conceptos se les facturarán:

    • ✅ Una Préstamo puede tener múltiples módulos
    • ✅ Los módulos determinan los conceptos aplicables
    • ✅ Se pueden modificar en cualquier momento
    • ✅ Permiten facturación diferenciada
    ✅ Ejemplo Práctico: Un apartamento puede tener los módulos "Residencial" + "Parqueadero", recibiendo tanto la cuota de Cuota como la cuota de parqueadero.

    ⚠️ Errores Comunes y Soluciones

    Error: "No tiene permisos para configurar"

    Problema: No puede acceder a la configuración del sistema.
    Causa: Su usuario no tiene permisos de administrador.
    Solución:
    • Contacte al administrador del sistema
    • Solicite permisos de configuración
    • Verifique que esté usando el usuario correcto

    Error: "Cuenta contable no válida"

    Problema: No se puede guardar la configuración contable.
    Causa: La cuenta seleccionada no existe o no está activa.
    Solución:
    • Verifique que la cuenta exista en el PUC
    • Confirme que la cuenta esté activa
    • Use cuentas de detalle (último nivel)
    • Consulte con el contador

    Error: "No se puede eliminar Línea"

    Problema: El sistema impide eliminar una Línea.
    Causa: La Línea tiene Préstamos asignadas.
    Solución:
    • Reasigne las Préstamos a otra Línea
    • O elimine las Préstamos primero
    • Verifique que no haya movimientos contables

    Problema: "Los cambios no se reflejan"

    Problema: Las modificaciones no aparecen en el sistema.
    Causa: Caché del navegador o cambios no guardados.
    Solución:
    • Actualice la página (F5)
    • Limpie la caché del navegador
    • Verifique que haya guardado los cambios
    • Cierre sesión y vuelva a ingresar
