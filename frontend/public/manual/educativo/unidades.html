🎯 Introducción

El módulo de Gestión de Estudiantes permite administrar todos los activos sujetos a cobro en su Gestión Educativa. Esto incluye Estudiantes, casas, locales comerciales, parqueaderos, depósitos y cualquier otro tipo de Estudiante que genere obligaciones de pago.

💡 Objetivo: Mantener un registro completo y actualizado de todas las Estudiantes, sus características, Acudientes y configuraciones de cobro.

¿Qué puede hacer en este módulo?

  • ✅ Crear y registrar nuevas Estudiantes
  • ✅ Editar información de Estudiantes existentes
  • ✅ Asignar Acudientes a las Estudiantes
  • ✅ Configurar Grados de participación
  • ✅ Gestionar módulos de cobro
  • ✅ Realizar acciones masivas en múltiples Estudiantes
  • ✅ Exportar listados a PDF
  • ✅ Filtrar y buscar Estudiantes
🌟 Novedad: Metamorfosis Multi-Sectorial

El sistema ahora es inteligente y se adapta a su sector. Si su entidad es un Colegio, los botones y títulos hablarán de "Estudiantes" y "Grados". Si es una Cooperativa de Transporte, hablarán de "Vehículos" y "Rutas". Todo de forma automática.

🚪 Cómo Acceder

  1. Ingrese al sistema ContaPY2
  2. Navegue al módulo "Gestión de Recaudos"
  3. Haga clic en "Estudiantes" o "Estudiantes / Activos"
📋 Nota: Dependiendo de la configuración de su sistema, esta opción puede aparecer como "Estudiantes", "Estudiantes / Activos" o simplemente "Estudiantes".

🏗️ Módulos de Cobro

Los módulos permiten aplicar diferentes conceptos de cobro según el tipo de Estudiante:

  • Residencial: Para Estudiantes y casas
  • Comercial: Para locales comerciales
  • Parqueadero: Para espacios de estacionamiento
  • Depósito: Para bodegas y cuartos útiles
  1. Seleccione los módulos que aplican para esta Estudiante
  2. Los módulos determinan qué conceptos de cobro se aplicarán
  3. Puede modificar los módulos posteriormente

Validaciones del Sistema

⚠️ Validaciones Importantes:
  • El código de Estudiante debe ser único en el sistema
  • El Grado debe ser un número decimal positivo
  • La suma de todos los Grados debería ser 100%
  • Todos los campos marcados como requeridos deben completarse

✏️ Editar Estudiante Existente

  • ✅ Grado o grupo

⚠️ Precaución al Editar:

  • Modificar el Grado afecta futuros cobros
  • Cambiar módulos puede alterar los conceptos aplicables

  2. Busque y seleccione el nuevo Acudiente

🗑️ Eliminar Estudiantes

Proceso de Eliminación

  1. Localice la Estudiante que desea eliminar en el listado
  2. Haga clic en el icono de eliminar (🗑️) en la columna de acciones
  3. Confirme la eliminación en el diálogo que aparece
  4. La Estudiante será removida del sistema
⚠️ IMPORTANTE - Restricciones de Eliminación:

No podrá eliminar una Estudiante si:

  • Tiene movimientos contables asociados
  • Tiene facturas generadas
  • Tiene pagos registrados
  • Tiene saldo pendiente

Solución: En estos casos, considere marcar la Estudiante como "Inactiva" en lugar de eliminarla.

Alternativa: Desactivar Estudiante

Si no puede eliminar la Estudiante por restricciones del sistema:

  1. Edite la Estudiante
  2. Cambie el estado a "Inactiva" (si está disponible)
  3. O agregue un prefijo como "INACTIVA-" al código
  4. Esto mantendrá el historial pero la excluirá de procesos futuros

⚡ Acciones Masivas

¿Qué son las Acciones Masivas?

Las acciones masivas permiten modificar múltiples Estudiantes simultáneamente, ahorrando tiempo cuando necesita hacer cambios similares en varias Estudiantes.

Proceso de Acciones Masivas

  1. Seleccionar Estudiantes: Use los checkboxes para marcar las Estudiantes que desea modificar
  2. Activar Modo Masivo: Aparecerá un botón "Acciones Masivas" cuando tenga Estudiantes seleccionadas
  3. Configurar Cambios: En el modal que aparece, configure qué módulos agregar o remover
  4. Aplicar Cambios: Confirme la operación para aplicar los cambios

Tipos de Acciones Masivas Disponibles

🏗️ Gestión de Módulos

La acción masiva más común es la gestión de módulos de cobro:

Acción Descripción Ejemplo de Uso
🟢 Asignar Módulo Agregar un módulo a las Estudiantes seleccionadas Agregar módulo "Parqueadero" a Estudiantes
🔴 Remover Módulo Quitar un módulo de las Estudiantes seleccionadas Remover módulo "Comercial" de Estudiantes
🔄 Combinado Asignar y remover módulos en la misma operación Cambiar de "Residencial" a "Comercial"

Interfaz de Acciones Masivas

  1. Selección Visual: Haga clic en los módulos para cambiar su estado:
    • 🔘 Gris: Sin cambios
    • 🟢 Verde: Asignar módulo
    • 🔴 Rojo: Remover módulo
  2. Confirmación: Revise el resumen antes de aplicar
  3. Aplicación: Los cambios se aplican inmediatamente
✅ Ejemplo Práctico: Si necesita agregar el módulo "Parqueadero" a 50 Estudiantes, seleccione los 50 Estudiantes, active acciones masivas, marque "Parqueadero" en verde y aplique. ¡Listo en segundos!

Consejos para Acciones Masivas

  • 💡 Use filtros para seleccionar Estudiantes similares más fácilmente
  • 💡 Revise siempre el resumen antes de confirmar
  • 💡 Haga pruebas con pocas Estudiantes primero
  • 💡 Mantenga respaldos antes de cambios masivos importantes

⚠️ Errores Comunes y Soluciones

Error: "Código de Estudiante ya existe"

Problema: Al crear una Estudiante, aparece el mensaje "El código ya existe".
Causa: Está intentando usar un código que ya está asignado a otra Estudiante.
Solución:
  • Verifique el listado de Estudiantes existentes
  • Use un código diferente y único
  • Considere usar un formato estándar (ej: APT-101, LOC-001)

Error: "Grado inválido"

Problema: El sistema no acepta el Grado ingresado.
Causa: El formato del Grado no es correcto.
Solución:
  • Use formato decimal: 0.8500 (no 0.85% ni 85%)
  • Use punto como separador decimal, no coma
  • Asegúrese de que sea un número positivo

Error: "No se puede eliminar la Estudiante"

Problema: El sistema impide eliminar una Estudiante.
Causa: La Estudiante tiene movimientos contables o facturas asociadas.
Solución:
  • Verifique si tiene saldos pendientes
  • Consulte el historial de movimientos
  • Considere desactivar en lugar de eliminar
  • Contacte al administrador si es necesario eliminar

Problema: "Las acciones masivas no funcionan"

Problema: No aparece el botón de acciones masivas.
Causa: No ha seleccionado ninguna Estudiante.
Solución:
  • Seleccione al menos una Estudiante usando los checkboxes
  • El botón aparecerá automáticamente
  • Verifique que tenga permisos para acciones masivas

Problema: "La búsqueda no encuentra Estudiantes"

Problema: El filtro de búsqueda no muestra resultados.
Solución:
  • Verifique que esté escribiendo correctamente
  • La búsqueda requiere al menos 2 caracteres
  • Limpie los filtros y intente de nuevo
  • Actualice la página si persiste el problema
