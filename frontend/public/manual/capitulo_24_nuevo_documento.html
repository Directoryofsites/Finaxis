Captura de Documentos Contables

Guía Maestra para la Gestión Integral de Movimientos en Finaxis v3.0

1. El Corazón Financiero

Esta página es el punto de entrada principal para toda la información financiera de su empresa. Aquí se consolidan desde simples asientos de ajuste hasta complejas liquidaciones de cartera y facturación electrónica. El diseño está optimizado para velocidad, precisión y cumplimiento legal.

2. La Cabecera: El ADN del Documento

Cada campo en la parte superior define la naturaleza contable y fiscal del registro. Es vital diligenciarlos correctamente para asegurar reportes íntegros.

Fecha Contable Obligatorio
Define el periodo fiscal donde se registrarán los movimientos. El sistema valida automáticamente que el mes no esté cerrado.
Tip: Puedes usar el calendario o digitar la fecha directamente. Presiona Enter para saltar al siguiente campo.
Tipo de Documento Obligatorio
Clasifica el origen de la transacción. Dependiendo de su elección, Finaxis activará funciones especiales (ej: si elige RC, se activará el módulo de cartera).
Dato Clave: La elección del tipo determina si el documento es informativo, fiscal o electrónico (DIAN).
Número / Consecutivo Dinámico
El sistema asigna automáticamente el número siguiente si el tipo es de "Numeración Automática". Si es "Manual" (ej: Facturas de Compra), usted deberá digitar el número del soporte físico.
Seguridad: Si el campo está bloqueado (gris), significa que el sistema garantiza la secuencia legal sin errores.
Tercero / Beneficiario Crítico
Identifica la persona o entidad asociada al movimiento. Es fundamental para la generación de medios magnéticos y estados de cuenta.
¡No te detengas!: Si el tercero no existe, usa el botón + RÁPIDO para crearlo sin salir de la página. El sistema lo seleccionará automáticamente al terminar.
Centro de Costo Opcional
Permite segmentar los gastos e ingresos por departamentos o proyectos. Si su empresa maneja contabilidad por áreas, este campo es indispensable.

3. Plantillas: Tu Tiempo Vale

¿Haces la misma nómina o arriendo cada mes?

Usa la casilla "Cargar Plantilla". El sistema llenará automáticamente todas las cuentas y conceptos. Tú solo ajustas los valores y guardas. ¡Un proceso de 5 minutos reducido a 30 segundos!

Nota: Al terminar un asiento complejo, puedes guardarlo como plantilla con el botón inferior para usarlo en el futuro.

4. Detalle Contable: Flujo Maestro

La grilla de movimientos es donde ocurre la magia. Aquí aplicamos la lógica de partida doble con herramientas de asistencia inteligente.

  • Búsqueda de Cuentas: Digita el código (ej: 1105) o el nombre de la cuenta. El sistema filtrará solo cuentas que permiten movimiento. Si la cuenta no existe, usa el ícono + azul para crearla en el acto.
  • Gestión de Conceptos: El primer concepto digitado se copiará a las siguientes líneas (efecto espejo). Además, puedes guardar conceptos favoritos con el ícono de disco 💾 para no volver a escribirlos nunca más.
    Truco Tab: Si dejas el concepto vacío y presionas Tab, el sistema pondrá automáticamente el nombre de la cuenta contable como descripción.
  • Digitación de Valores: Olvida los ceros.
    • Presiona + para agregar 000 (miles).
    • Presiona - para agregar 00 (centenas).
    • Ejemplo: Escribe 15 y presiona +. El sistema pondrá 15,000 instantáneamente.
  • El Cuadre Automático: En la última fila de tu asiento, presiona Tab sobre el campo Débito o Crédito vacío. El sistema calculará la diferencia exacta para que el documento quede en Sumas Iguales y te llevará directo al botón de Guardar.

5. Especial de Tesorería (Cruce de Facturas)

Al hacer Recibos de Caja o Pagos a Proveedores, aparecerá una tabla adicional con los saldos pendientes del tercero.

Lógica de Cruce Inteligente
1. Digita el Valor Total a Abonar en la cabecera.
2. En la tabla de facturas, ubícate en el campo "Abono".
3. Presiona Tab. El sistema aplicará automáticamente el saldo disponible y saltará a la siguiente factura.
Indicador Visual: El total aplicado se verá en Verde si coincide exactamente con el abono, o en Rojo si hay una diferencia pendiente por aplicar.

6. Monitor de Operaciones (Control Total)

A la derecha de su pantalla encontrará el Monitor de Asientos. Es su asistente de supervisión en tiempo real, diseñado para que no pierda rastro de lo que acaba de registrar.

Filtros Inteligentes
Localice rápidamente cualquier documento por Fecha, Tipo, Número o Beneficiario. Ideal para auditorías rápidas antes de cerrar el día.
Edición "On-The-Fly"
¿Cometió un error? No busque más. En el monitor, haga clic directamente sobre el Código del Documento (ej: CC 000123). El sistema abrirá la edición para que corrija valores o conceptos de inmediato.
📋
Acciones Multitarea:
• 🖨️ Imprimir: Obtenga el PDF de cualquier documento histórico con un clic.
• 📄 Exportar: Descargue la relación completa de movimientos filtrados a PDF.
• 🔍 Auditoría de Cuenta: Haga clic en el Código de Cuenta dentro del monitor para saltar directo al Reporte Auxiliar de esa cuenta específica.

7. Finalización y Cumplimiento

Guardar Documento
Registra la operación en los libros oficiales de forma inmediata.
Guardar y Emitir DIAN
Para Facturación Electrónica o Documento Soporte. Valida la información y envía los datos a la DIAN en tiempo real, obteniendo el código CUFE/CUDE.
Botón Imprimir Post-Guardado
Una vez guardado el documento, aparecerá un botón resaltado con el número del documento. Al pulsarlo, se generará el PDF oficial listo para su entrega o archivo.
