Manual de Usuario: Libro Diario Resumido

Bienvenido al módulo del Libro Diario Resumido. Este reporte es una herramienta fundamental para la revisión contable y la auditoría rápida de movimientos.

¿Para qué sirve?

A diferencia del Libro Diario tradicional que muestra cada transacción individual, este reporte agrupa los movimientos por Tipo de Documento (ej: Facturas, Recibos, Nómina). Esto le permite verificar en segundos los totales operados por módulo sin perderse en miles de registros.

Características Principales

  • Agrupación Automática: Los asientos contables se ordenan por su origen (Tipo de Documento).
  • Control Integridad Contable: El sistema totaliza los Débitos y Créditos de cada grupo. Si un documento está descuadrado, lo notará inmediatamente porque las sumas no coincidirán.
  • Filtros Flexibles: Puede seleccionar un periodo de tiempo y filtrar por tipos de documentos específicos.

Cómo generar el reporte

  1. Ingrese al menú Contabilidad > Reportes > Libro Diario Resumen.
  2. Seleccione el Rango de Fechas: Por defecto muestra el año en curso.
  3. Filtre por Tipo de Documento:
    • Si no selecciona nada, el sistema mostrará TODOS.
    • Puede seleccionar "Seleccionar Todos" para marcar todo rápidamente.
    • Use Ctrl + Click para seleccionar varios documentos específicos (ej: solo "Facturas de Venta" y "Recibos de Caja").
  4. Haga clic en el botón Generar Resumen.

Interpretación de Resultados

El reporte le mostrará una tabla dividida por secciones:

  • Encabezado de Sección: Nombre del Tipo de Documento (ej: "RECIBO DE CAJA").
  • Cuerpo: Lista de cuentas afectadas, mostrando el Código y Nombre de la cuenta.
  • Totales: Suma de Débitos y Créditos para ese tipo de documento.

Exportación

Utilice los botones en la parte superior derecha de los resultados:

  • PDF: Genera un archivo listo para impresión en una nueva pestaña (Ideal para archivo físico).
  • CSV: Descarga un archivo plano compatible con Excel para análisis de datos avanzado.
