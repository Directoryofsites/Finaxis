Manual de Operaciones: Desarrollo con Doble IA

Guía paso a paso para coordinar el trabajo entre el Usuario, Antigravity y Kiro.

PARA TI (EL USUARIO)

1. Configuración del Entorno de Trabajo

Sigue estos pasos para preparar las carpetas antes de invocar a las IAs.

PASO 1: Clonar la Carpeta
Copia toda tu carpeta actual del proyecto (c:\ContaPY2) y pégala en otra ubicación o con otro nombre, por ejemplo: c:\ContaPY2_Kiro.
  • Carpeta A (Original): Aquí trabajará la IA Antigravity.
  • Carpeta B (Copia): Aquí trabajará la IA Kiro.
PASO 2: Crear Ramas Git Separadas
Es vital que cada IA trabaje en su propia línea de tiempo (rama) para no chocar.

En la Carpeta A (Antigravity):

git checkout -b antigravity-rama

En la Carpeta B (Kiro):

git checkout -b kiro-rama
Importante: Nunca trabajes directamente en main mientras estás desarrollando en paralelo.
PARA LA IA "KIRO" - COPIA Y PEGA ESTO EN SU CHAT

2. Instrucciones de Configuración para Kiro

(Usuario: Copia todo el texto de abajo y pégaselo a Kiro en su chat).

Hola Kiro, soy tu compañero humano.

Estamos trabajando en un entorno paralelo. Antigravity está en la carpeta original. Tú trabajarás en esta carpeta clonada.

Por favor, realiza las siguientes configuraciones AHORA MISMO para que podamos ejecutar ambos proyectos simultáneamente sin conflictos de puertos:

A. Configura el Backend (Python)

Edita el archivo run.py en la raiz y cambia el puerto a 8002:

# Archivo: run.py
if __name__ == "__main__":
    os.environ['GIO_USE_VOLUME_MONITOR'] = 'dummy'
    # CAMBIO: Puerto 8002
    uvicorn.run("app.main:app", host="127.0.0.1", port=8002, reload=True, reload_dirs=["app"])

B. Configura el Frontend (Next.js)

1. Edita el archivo frontend/.env.local y apunta al nuevo backend:

# Archivo: frontend/.env.local
NEXT_PUBLIC_API_URL=http://localhost:8002

2. Cuando inicies el servidor de desarrollo, usa este comando para el puerto 3002:

npm run dev -- -p 3002

(O modifica el script "dev" en package.json para incluir `-- -p 3002`).

Tu Misión:

Una vez configures esto, inicia los servidores. Deberías estar corriendo en:

  • Frontend: http://localhost:3002
  • Backend: http://localhost:8002

Antigravity estará corriendo en los puertos 3000 y 8000.

PARA TI (EL USUARIO)

3. Rutina de Cierre del Día (Unificación)

Al terminar la sesión, sigue estos pasos para juntar el trabajo de ambas IAs.

1. Guardar cambios en ambas carpetas (Commit).
2. En la Carpeta A (Antigravity): 
git add .
git commit -m "Avances Antigravity dia X"
git checkout main
git merge antigravity-rama
3. En la Carpeta B (Kiro): 
git add .
git commit -m "Avances Kiro dia X"
# Empujar la rama al repositorio remoto (si usas GitHub)
git push origin kiro-rama
4. Volver a Carpeta A y traer cambios de Kiro: 
# Descargar cambios remotos
git fetch origin
# Fusionar rama de Kiro en main
git merge origin/kiro-rama

Resultado final: La rama main tendrá el código de ambas IAs.
